## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/jarvis-collab/invoice-generator.git
   cd invoice-generator
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. **Enter Business Details** - Add your company information
2. **Add Client Information** - Include client/customer details
3. **Add Items/Services** - List products or services with quantities and rates
4. **Set Tax Rate** - Add applicable tax percentage (optional)
5. **Add Notes** - Include payment terms or thank you messages
6. **Generate PDF** - Download your professional invoice instantly
